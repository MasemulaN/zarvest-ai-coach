# Zarvest 💸  
### *“Understand your money. Improve your future.”*

Zarvest is an AI-powered financial intelligence platform designed to help users better understand their spending habits, improve budgeting, and make smarter financial decisions using AI-driven insights.

Built using Loveable through prompt engineering and AI-assisted development.

---

## 🚀 Features

- Upload CSV or PDF bank statements
- AI-powered transaction categorization
- Interactive financial dashboard
- Financial personality analysis
- AI-generated budgeting insights
- Expense forecasting & spending trends
- Hidden subscription & money leak detection
- What-if financial simulator
- AI Financial Coach Chat Assistant
- South African Rand (ZAR) support

---

# 🧠 Financial Personality Engine

Zarvest analyzes user behavior and assigns financial personality profiles such as:

- Careful Saver
- Balanced Planner
- Impulse Spender
- Living-Large Spender

The system evaluates:
- spending behavior,
- income consistency,
- savings habits,
- and transaction patterns

to improve analysis accuracy.

---

# 📊 Dashboard Features

The dashboard includes:
- Spending breakdown charts
- Monthly financial trends
- Savings projections
- Budget recommendations
- Risk alerts
- AI-generated financial insights

The interface follows a modern fintech-style design with light and dark mode support.

# 🛠️ Technology Stack

Zarvest was developed using Loveable, an AI-powered development platform that enables applications to be built through prompt engineering and AI-assisted generation instead of traditional coding from scratch.

### Technologies Used
- React
- Tailwind CSS
- Node.js
- Financial chart libraries
- AI-powered financial analysis systems

The development process focused heavily on:
- prompt engineering,
- AI-assisted design,
- feature refinement,
- and intelligent financial analysis workflows.
